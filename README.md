🚀 Experience Windows 11… without installing it!

This project faithfully recreates the Windows 11 interface right in your browser, featuring smooth animations and a design inspired by Fluent Design.  
Open the Start menu, launch Edge, File Explorer, Settings, and more — all 100% web-based.

✨ Features:
- 🎨 Authentic Windows 11 look & feel
- 🖱️ Interactive Start menu and taskbar
- 📂 Simulated apps (browser, file explorer, settings…)
- ⚡ Smooth performance powered by modern web technologies
- 📱 Works on desktop, tablet, and mobile

🔍 **Note:** This project is a visual tribute to Windows 11 and is not affiliated with Microsoft.
